## 核心技术详解

### 1. Elo等级分系统 ⚽

**基础原理**：
- 基于eloratings.net历史数据初始化球队实力
- 东道主（墨/美/加）全程+60主场加成
- Elo差距 → 胜负期望：`We = 1 / (1 + 10^((elo_b - elo_a) / 400))`

**参数配置**：
| 参数 | 值 | 说明 |
|------|-----|------|
| HOST_ELO_BONUS | 60 | 东道主主场加成 |
| BASE_TOTAL_GOALS | 2.8 | 势均力敌时预期总进球 |
| TOTAL_MISMATCH | 2.5 | 实力悬殊时总进球上浮 |
| GD_LINEAR | 3.2 | 净胜球线性项 |
| GD_CUBIC | 12.0 | 极端比分非线性项 |

---

### 2. 双泊松分布模型 📊

**进球生成**：
```
总进球 = 基础值 + 实力悬殊加成
净胜球 = 线性项 + 非线性项
```

**泊松分布**：
- 两队独立生成进球分布
- `P(X=k) = λ^k * e^(-λ) / k!`
- Dixon-Coles修正：上调0-0/1-1等低比分概率（避免低估平局）

---

### 3. 知识库融合机制 📚

**数据来源**：
| 来源 | 内容 | 权重 |
|------|------|------|
| TXT球队库 | 近10场战绩、胜率、场均进球/失球、战术风格 | 70% |
| EPUB/DOCX | 正面/负面评价、爆冷风险、战术描述 | 30% |

**融入方式**：

**A. Elo修正（effective_elo）**
```
delta += (form - 1.0) * 80.0           # 基础修正
delta -= strong_but_weak * 80.0         # 强队不稳
delta += giant_killer * 60.0            # 遇强不弱
delta -= strong_caution * 150.0         # 全局强队下调
delta += upset_emphasis * 120.0         # 全局弱队上调
```
修正上限：±30 Elo

**B. 攻防因子**
```python
home_adj = attack_factor / defense_factor  # 范围 0.70~1.40
away_adj = 同理
```

**C. 风格开放度**
```python
style = sqrt(open_a * open_b)  # 攻势足球=大球，保守型=小球
```

### 4. 赔率市场融合 💰

**数据来源**：
| 来源 | 说明 | 优先级 |
|------|------|--------|
| the-odds-api.com | 国际博彩公司赔率 | 最高 |
| sporttery.cn | 中国体彩网竞彩赔率 | 其次 |
| Elo推断 | 无外部数据时兜底 | 最后 |

**胜负期望融合**：
```python
we_blend = 0.7 * we_model + 0.3 * we_market
```

**大小球偏差（Score Shift）**：
```python
if 市场期望进球 - 模型期望进球 > 0.2:
    调整：提高大比分概率（最多转移12%）
elif 市场期望进球 - 模型期望进球 < -0.2:
    调整：提高小比分概率
```

**冷门风险检测**：
```python
upset_risk = |模型胜率 - 市场胜率| / 模型胜率
```
最大风险：30%

### 5. 激进模式加成 🚀

**触发条件**：实力差距大时（Elo差距>150）

**加成机制**：

| 加成类型 | 公式 | 效果 |
|----------|------|------|
| 进球期望加成 | +20% | 强队期望进球增加 |
| 零封加分 | 0.025+0.015×Elo差 | 强队零封比分加分 |
| 大比分加分 | 0.012×(进球数-2) | 3-0/4-0等加分 |
| 期望匹配权重 | 加倍 | 比分选择更激进 |
